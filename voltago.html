Über das Projekt

Was ist VoltaGo?

VoltaGo ist ein kommandozeilenbasiertes Python Programm welches Ihnen als professioneller Anwender dabei hilft, verschiedene Berechnungen Ihres Arbeitsalltages zuverlässig zu übernehmen. Mit VoltaGo 3 können Sie die Beladung von Lastkraftwagen und die ideale Belegung eines Daches jeder Größe mit Photovoltaikmodulen berechnen. Der Fokus liegt hierbei auf dem PV Programm, an welchem stetig weiterentwickelt wird. VoltaGo ist selbst als Konsolenprogramm einfach und unmissverständlich zu bedienen, es sagt Ihnen immer in verständlicher Sprache welche Informationen es gerade von Ihnen braucht und wie Sie so antworten können, dass VoltaGo Sie versteht.

Support & Anleitung
Bedienungsanleitung

Erste Schritte: Starten Sie die VoltaGo.exe. Beim ersten Start werden Sie aufgefordert, die Lizenzvereinbarung zu akzeptieren und bekommen die Updateinformationen Ihrer Version angezeigt. Sie landen nun in der Haupteingabeshell von VoltaGo. Wo Sie sich im System befinden, sprich welche spezifischen Befehle erkannt werden können, können Sie unten links sehen. In der Hauptshell können Sie mit -list die Befehlsliste abrufen. Geben Sie einfach einen Befehl ein und drücken Sie die ENTER Taste, VoltaGo sagt Ihnen dann was es tun wird.

Update-Prozess: Um die neueste Version zu erhalten, nutzen Sie in der App den Befehl -update. VoltaGo verbindet sich mit den BLWE Servern und informiert Sie wenn eine neue Version verfügbar ist, die Sie hier laden können.

Troubleshooting / Fehlerbehebung

Problem: Programm startet nicht.
Stellen Sie sicher, dass die VoltaGo.exe Datei sich im Verzeichnis mit allen benötigten Zusatzdateien (.txt Dateien) befindet. Ohne diese kann VoltaGo nicht starten.

Problem: Es wird eine Fehlermeldung bezüglich meiner Internetverbindung angezeigt.
Prüfen Sie Ihre Internetverbindung. VoltaGo prüft bei jedem Start und wenn Sie den Updatebefehl ausführen ob eine neue Version verfügbar ist und benötigt dafür eine Verbindung zum Internet. Sind die BLWE Server nicht erreichbar, gibt VoltaGo eine Fehlermeldung aus. Sie können das Programm jedoch ohne Einschränkungen verwenden, haben möglicherweise aber nicht die neuesten Funktionen.

Problem: Programm stürzt ab.
Sollte das Programm in einem Unterprogramm wie dem PV-Programm abstürzen, haben Sie wahrscheinlich bei der Abfrage nach einer Zahl eine Zahl im falschen Format oder keine gültige Eingabe getätigt. Das Programm kann dies aktuell nicht verarbeiten. Sollte das Programm abgestürzt sein wenn dies nicht der Fall ist kontaktieren Sie bitte den Entwickler und geben Sie einen eventuell angezeigten Fehlercode an. Dies sollte nicht passieren.

Problem: Programm startet im Legacybootmodus
Überprüfen Sie die Einstellung Ihres Systemdatums. Wenn VoltaGo ein Datum zu weit in der Zukunft erkennt sperrt es aus Sicherheitsgründen den Zugang zum Programm. Sollte Ihre Version jedoch älter als ein halbes Jahr sein, so benötigen Sie einen Legacybootcode zum Start des Programms. Diesen bekommen Sie auf dieser Seite. Jedoch sollten Sie ein Update des Programms strengstens in Betrachtung ziehen da der Zugang aufgrund eventuell mittlerweile behobener schwerwiegender Bugs eingeschränkt wurde.
